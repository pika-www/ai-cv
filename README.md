# AI CV Studio

一个给自己和朋友使用的 AI 简历编辑器。用户可以上传旧简历或粘贴简历文本，让 AI 分析并在右侧给出优化建议；用户可以直接编辑简历，也可以输入新增经历让 AI 自动放入合适位置，最后导出同版式 PDF。

## 当前前端覆盖内容

- 中文 / 英文界面切换
- 上传旧简历入口
- 粘贴简历文本入口
- AI 优化建议面板
- 新增经历自动插入
- 可编辑简历预览
- PDF 导出预览

这是第一版产品界面。当前数据仍然是 React 内部模拟数据，后续会接入后端 API。

## 本地运行

```bash
npm install
npm run dev
```

默认地址：

```bash
http://localhost:5173/
```

## 构建

```bash
npm run build
```

## 架构方向

这个项目先按网页 SaaS 的方式开发，同时保留未来开源或本地部署的可能性：

- AI prompt 和模型 provider 放在后端 API 后面，不直接写死在前端。
- 用户可见文案保持结构化，方便继续完善 i18n。
- 导出重点是与预览一致的 PDF，Markdown 只作为中间格式。
- AI 建议和新增经历插入必须基于用户上传、粘贴或输入的内容。
